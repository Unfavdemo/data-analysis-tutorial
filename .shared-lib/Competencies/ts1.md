# TS.1 Core Programming - Complete Framework

## **Competency Framework: Technical Skills**

## **Domain: Core Programming (TS.1)**
**Description**: I can apply core programming principles

---

## **Competency 1: Create and manipulate variables (TS.1.1)**
**Description**: I can store values as variables and promptly use them

### **Skill 1.1: Declare variables (TS.1.1.1) - Level 1**
**Indicator**: Can declare variables with appropriate data types
**Assessment**: Student can create variables using correct syntax for strings, integers, floats, and booleans

### **Skill 1.2: Assign variable values (TS.1.1.2) - Level 1**  
**Indicator**: Can assign and reassign variable values correctly
**Assessment**: Student can initialize variables and update their values throughout a program

### **Skill 1.3: Use variables in operations (TS.1.1.3) - Level 2**
**Indicator**: Can use variables in calculations and operations
**Assessment**: Student can perform mathematical operations, string concatenation, and logical operations using variables

---

## **Competency 2: Understand statements (TS.1.2)**
**Description**: I can create statements to programmatically perform tasks

### **Skill 2.1: Write sequential statements (TS.1.2.1) - Level 1**
**Indicator**: Can create statements that execute in logical sequence
**Assessment**: Student can write code where statements execute line-by-line in proper order

### **Skill 2.2: Implement program flow (TS.1.2.2) - Level 2**
**Indicator**: Can create statements that control program execution
**Assessment**: Student can structure code with proper indentation and control flow understanding

### **Skill 2.3: Debug statements (TS.1.2.3) - Level 2**
**Indicator**: Can identify and fix errors in program statements
**Assessment**: Student can locate syntax errors, logic errors, and runtime errors in code

---

## **Competency 3: Utilize conditionals (TS.1.3)**
**Description**: I can use conditionals to check for and parse different outcomes

### **Skill 3.1: Create if/else statements (TS.1.3.1) - Level 2**
**Indicator**: Can implement basic conditional logic with if/else
**Assessment**: Student can write simple decision-making code using if, elif, and else statements

### **Skill 3.2: Use comparison operators (TS.1.3.2) - Level 2**
**Indicator**: Can apply comparison and logical operators effectively
**Assessment**: Student can use ==, !=, <, >, <=, >=, and, or, not in conditional statements

### **Skill 3.3: Build complex conditionals (TS.1.3.3) - Level 3**
**Indicator**: Can create nested and compound conditional statements
**Assessment**: Student can build multi-layered decision trees and complex boolean logic

---

## **Competency 4: Optimize statements (TS.1.4)**
**Description**: I can use loops and user-defined functions to optimize my code

### **Skill 4.1: Implement loops (TS.1.4.1) - Level 2**
**Indicator**: Can create and use for/while loops to reduce code repetition
**Assessment**: Student can identify repetitive tasks and implement appropriate loop structures

### **Skill 4.2: Create functions (TS.1.4.2) - Level 2**
**Indicator**: Can define and call user-defined functions with parameters
**Assessment**: Student can write reusable code blocks with parameters and return values

### **Skill 4.3: Apply optimization principles (TS.1.4.3) - Level 3**
**Indicator**: Can identify and eliminate redundant code using DRY principles
**Assessment**: Student can refactor code to improve efficiency and maintainability

---

**Total TS.1 Structure:**
- 4 Competencies
- 12 Skills (4 Level 1, 6 Level 2, 2 Level 3)
- Progressive difficulty from foundational to advanced concepts
