# TS.2 Front-End - Complete Framework

## **Domain: Front-End (TS.2)**
**Description**: I can create a front-end to an application

---

## **Competency 1: Design a user experience (TS.2.1)**
**Description**: I can design a user experience that meets the needs of users

### **Skill 1.1: Research user needs (TS.2.1.1) - Level 1**
**Indicator**: Can identify target audience and their requirements
**Assessment**: Student can conduct basic user research and create user personas

### **Skill 1.2: Create user journey maps (TS.2.1.2) - Level 2**
**Indicator**: Can map user interactions and touchpoints
**Assessment**: Student can design user flow diagrams showing how users navigate through an application

### **Skill 1.3: Apply UX principles (TS.2.1.3) - Level 3**
**Indicator**: Can implement advanced UX design principles
**Assessment**: Student can apply accessibility, usability, and user-centered design principles

---

## **Competency 2: Create a wireframe (TS.2.2)**
**Description**: I can plan and sketch the layout, functionality, and content of a web page

### **Skill 2.1: Sketch basic layouts (TS.2.2.1) - Level 1**
**Indicator**: Can create simple wireframe sketches
**Assessment**: Student can draw basic page layouts showing content placement and structure

### **Skill 2.2: Design responsive layouts (TS.2.2.2) - Level 2**
**Indicator**: Can plan layouts for multiple screen sizes
**Assessment**: Student can create wireframes that adapt to desktop, tablet, and mobile viewports

### **Skill 2.3: Prototype interactions (TS.2.2.3) - Level 3**
**Indicator**: Can create interactive wireframe prototypes
**Assessment**: Student can build clickable prototypes showing user interactions and navigation

---

## **Competency 3: Build a front-end (TS.2.3)**
**Description**: I can create an interactive product, including buttons, forms, and navigation

### **Skill 3.1: Structure HTML content (TS.2.3.1) - Level 1**
**Indicator**: Can create semantic HTML markup
**Assessment**: Student can write clean, semantic HTML with proper tags and structure

### **Skill 3.2: Style with CSS (TS.2.3.2) - Level 2**
**Indicator**: Can apply CSS styling and layout techniques
**Assessment**: Student can use CSS for styling, flexbox/grid layouts, and responsive design

### **Skill 3.3: Add JavaScript interactivity (TS.2.3.3) - Level 3**
**Indicator**: Can implement interactive features with JavaScript
**Assessment**: Student can create dynamic content, handle events, and manipulate the DOM

---

## **Competency 4: Utilize front-end tools (TS.2.4)**
**Description**: I can choose and use the right tools to build and deploy front-end applications

### **Skill 4.1: Select appropriate frameworks (TS.2.4.1) - Level 2**
**Indicator**: Can evaluate and choose suitable front-end frameworks
**Assessment**: Student can compare frameworks like React, Vue, or Angular and select appropriate tools

### **Skill 4.2: Use development tools (TS.2.4.2) - Level 2**
**Indicator**: Can utilize browser developer tools and build processes
**Assessment**: Student can debug using DevTools, use package managers, and build tools

### **Skill 4.3: Deploy applications (TS.2.4.3) - Level 3**
**Indicator**: Can deploy front-end applications to production
**Assessment**: Student can use hosting platforms, configure domains, and manage deployment pipelines

---

**Total TS.2 Structure:**
- 4 Competencies  
- 12 Skills (3 Level 1, 6 Level 2, 3 Level 3)
- Progressive difficulty from basic design concepts to advanced deployment
